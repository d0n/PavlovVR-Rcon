# Summary
Simple Rcon for Pavlov VR

# Config
Edit server.json with array of selected servers, passwords must be MD5
```
[{
        "ip": "1.1.1.1",
        "port": "9100",
        "password": "098f6bcd4621d373cade4e832627b4f6"
    },
    {
        "ip": "2.2.2.2",
        "port": "8082",
        "password": "098f6bcd4621d373cade4e832627b4f6"
    }
]
```
# Usage
```node init```
